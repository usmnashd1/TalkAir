TalkAir 🌐🎧

A lightweight browser-based talk-to-strangers voice chat app hosted on GitHub Pages.

No accounts. No installs. Just open and connect.

🚀 What it does

TalkAir randomly connects you with another online user for a real-time voice chat using WebRTC.

Anonymous 1v1 voice chat
Random matchmaking
Instant connect / disconnect
Works directly in browser
No login required
🧠 How it works
Uses WebRTC for peer-to-peer audio
Uses a simple signaling server (WebSocket / Worker) to match users
Once connected, audio flows directly between browsers (no media server in between)
🛠️ Tech Stack
HTML, CSS, JavaScript
WebRTC (audio streams)
WebSocket / Cloudflare Worker (signaling / pairing)
GitHub Pages (frontend hosting)
▶️ How to use
Open the site
https://usmnashd1.github.io/TalkAir/
Allow microphone access
Click Start / Connect
You get matched with a random stranger
Talk. Skip. Repeat.
